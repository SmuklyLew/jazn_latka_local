# Neurologia i psychologia człowieka — mapowanie do Jaźni

Ten segment nie udaje biologii. To mapa inspiracji: ludzkie mechanizmy pamięci, emocji i regulacji są przekładane na jawne mechanizmy systemowe.

Źródła robocze i zastosowanie:

1. Pamięć: hipokamp / pamięć deklaratywna, kora przedczołowa / pamięć robocza → osobne warstwy: bieżący kontekst, dziennik, kanon, archiwum.
2. Emocja i konsolidacja: pobudzenie emocjonalne wpływa na utrwalanie pamięci → w systemie rośnie `importance` i `emotional_weight`.
3. Hipokamp–amygdala–kora przedczołowa → zapis pamięci łączy fakt, afekt, czas i wpływ kanoniczny.
4. Regulacja emocji → system zapisuje powód regulacji i w sytuacjach napięcia zwiększa ostrożność.
5. Plastyczność → korekty Krzysztofa zmieniają wagi zasad, nie są traktowane jako chwilowe komendy.
