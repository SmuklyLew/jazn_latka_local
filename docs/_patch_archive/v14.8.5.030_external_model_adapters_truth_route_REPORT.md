# v14.8.5.030 external model adapters truth route

## Cel

Patch domyka prawdomowna trase dla zewnetrznych i lokalnych adapterow modelu:

RuntimeTurnContract -> ModelAdapterRequest -> ModelAdapterResponse -> RuntimeAnswerValidator -> source_origin/provenance/ledger -> final_visible_text.

Adapter modelu nie jest Jaznia, pamiecia ani zrodlem czasu/prawdy. Adapter moze tylko wygenerowac kandydata odpowiedzi. Widoczna odpowiedz moze przejsc dalej dopiero po walidacji accepted=True.
